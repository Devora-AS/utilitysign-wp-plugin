UtilitySign Signing Form Test

This page helps you test how the UtilitySign signing form appears to visitors.

How to Test the Signing Form:

  1. Create a WordPress Page:
    • Go to your WordPress admin: http://devora-test.local/wp-admin
    • Navigate to Pages → Add New
    • Title: "Test Signing Form"
  2. Add the Shortcode:
    [utilitysign_signing_form document_id="test-doc-123" enable_bank_id="true" enable_email_notifications="true"]
  3. Publish and View:
    • Publish the page
    • Visit the page as a visitor (not logged in)
    • Check how the form renders and behaves

Alternative Testing Methods:

  1. Gutenberg Block:
    • In the page editor, click the "+" button
    • Search for "UtilitySign Signing Form"
    • Add the block and configure it
  2. Widget Area:
    • Go to Appearance → Widgets
    • Add a "Shortcode" widget
    • Paste the shortcode above
  3. Theme Template:
    • Add the shortcode directly to a theme template file
    • Use: <?php echo do_shortcode('[utilitysign_signing_form document_id="test-doc-123"]'); ?>

What to Look For:

  • ✅ Form loads without JavaScript errors
  • ✅ Devora design system styling is applied
  • ✅ Form fields are properly labeled and styled
  • ✅ BankID integration button appears
  • ✅ Form validation works
  • ✅ Mobile responsiveness
  • ✅ Error handling displays properly
Available Shortcode Attributes:
• document_id (required): The document ID to sign
• enable_bank_id (optional): Enable/disable BankID (default: true)
• enable_email_notifications (optional): Enable/disable email notifications (default: true)
• class_name (optional): Additional CSS classes
