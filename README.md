# UtilitySign WordPress Plugin (Distribution)

This repository is **distribution-only** for the UtilitySign WordPress plugin.

- **Source of truth**: `Devora-AS/Devora-UtilitySign` under `wp-plugin/utilitysign/`
- **Releases**: created automatically from monorepo releases
- **Do not develop here**: open PRs against the monorepo instead

## Downloads
Install/upgrade using the ZIP assets attached to GitHub Releases in this repository.
